## [1.0.0] - 2026-06-28

### Added
- **Core Platform**: Initial release of the Sentrix Enterprise SOC Platform.
- **Frontend**: Next.js App Router UI with Tailwind CSS and React Query.
- **Backend**: FastAPI implementation with JWT authentication and RBAC.
- **Database**: PostgreSQL integrations via SQLAlchemy.
- **Modules**: Alert Management, Threat Intelligence, Malware Analysis, Threat Hunting, SOAR, Cases.
- **Infrastructure**: Docker Compose configuration for one-click deployment.
