# Sentrix Roadmap

Our vision is to make Sentrix the de-facto open-source SOC platform.

## Near Term (Q3 2026)
- **SIEM Integrations**: Native ElasticSearch and Splunk forwarder connectors.
- **Automated Reporting**: Scheduled PDF generation for CISO-level reports.
- **Enhanced AI**: Retrieval-Augmented Generation (RAG) for Threat Intel querying via Qdrant.

## Mid Term (Q4 2026)
- **Multi-Tenancy**: Support for MSSPs (Managed Security Service Providers) to manage isolated orgs.
- **Advanced SOAR**: Visual DAG builder for complex playbook execution.
- **Agent Deployment**: Lightweight OSSEC/Wazuh agents for endpoint telemetry.

## Long Term (2027)
- **Cloud-Native Deployments**: Helm charts and Kubernetes Operators for auto-scaling.
- **Marketplace**: Community-driven SOAR playbooks and detection rules.
